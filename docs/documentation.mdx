# Quick Start

There are several ways to start learning with LearnPack.

**Online (recomended)**

We have partnered with Gitpod, a cloud based IDE, to offer one-click support to all of the tutorials; You can start learning in a matter of seconds.

**Local Installation**

You can also download any tutorial to your local computer to avoid any interruption and/or use your local editor. 

1. Type the following command on your computer terminal:

```sh
$ npm install -g learnpack
```
